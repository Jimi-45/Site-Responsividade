# ✈️ Jornada Viagens - Alura

Uma landing page responsiva de viagens e comparação de preços de passagens, desenvolvida como projeto prático durante os meus estudos na plataforma **Alura**. O foco principal deste projeto foi aplicar os conceitos de **Mobile First** e otimização para motores de busca (**SEO**).

---

## 🎯 Objetivo do Projeto

O objetivo foi construir um site moderno, rápido e totalmente acessível a partir de qualquer dispositivo, priorizando a experiência em smartphones e garantindo que o código seguisse as melhores práticas de indexação do Google.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** (Semântico)
* **CSS3** (Flexbox e Grid Layout)
* **Mobile First Design** (Abordagem focada em dispositivos móveis)
* **SEO Local e Técnico** (Meta tags, acessibilidade e estruturação)

---

## 📱 Conceitos Aprendidos e Aplicados

### 1. Responsividade & Mobile First
Ao invés de criar o design para desktop e depois "esmagar" a tela para o celular, o projeto foi desenvolvido **primeiro para telas menores**. 
* Uso estratégico de `@media queries` para adaptar o layout progressivamente para tablets e desktops.
* Unidades de medida relativas (`rem`, `em`, `%`) para garantir a fluidez dos elementos e textos.

### 2. Otimização para SEO (Search Engine Optimization)
Para que o site de viagens seja facilmente encontrado nos motores de busca, apliquei:
* **HTML Semântico:** Uso correto de tags como `<header>`, `<main>`, `<section>`, `<nav>` e `<footer>`, ajudando os robôs de busca a entenderem a hierarquia da página.
* **Meta Tags:** Configuração de `meta description`, títulos estratégicos (`<h1>` a `<h3>`) e atributos `alt` em todas as imagens para acessibilidade e indexação.
* **Performance:** Imagens otimizadas para carregamento rápido em redes móveis.
